# local-server-tool
使用node.js 搭建简易的本地服务器  

## 环境搭建
需要安装node.js并安装模块  
echo * npm install express -g  
echo * npm install supervisor -g

## 启动
双击 rserver.bat 文件  
端口设置默认为7600
